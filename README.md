# 🎙️ Sonal Kaushal Portfolio Website

This project is a **responsive portfolio website dedicated to Sonal Kaushal**, a well-known Indian voiceover artist who has voiced many popular animated characters.

The website highlights her **voice acting work, personal background, and professional journey**, allowing visitors to explore the cartoons she has voiced and learn more about her career.

The project was built using **HTML, CSS, and Bootstrap 5**, focusing on creating a **clean, responsive, and user-friendly interface** that works smoothly across different devices such as desktops, tablets, and mobile phones.

## ✨ Features

🎤 **Voiceover Work Showcase**
The website displays the cartoons and characters that Sonal Kaushal has provided voiceovers for. This section helps visitors quickly explore her work in the animation and dubbing industry.

👩 **About Section**
Provides detailed information about Sonal Kaushal, her background, and her journey as a voiceover artist.

📱 **Responsive Design**
The website is fully responsive and adapts to different screen sizes using **Bootstrap 5**, ensuring a smooth user experience across all devices.

📩 **Contact Section**
A dedicated contact section allows visitors or potential collaborators to easily reach out.

🎨 **Clean and Simple UI**
The website focuses on a minimal and organized design that highlights the content while maintaining a professional appearance.
